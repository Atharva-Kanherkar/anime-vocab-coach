# Site atmosphere assets (optional)

The homepage hero uses a **CSS gradient atmosphere** by default. No image required.

To add an AI-generated or licensed abstract background (recommended: blurred cityscape,
soft sky, bokeh lights; **no recognizable characters**):

1. Export as `hero-atmosphere.webp` (~1920×1080, dark, heavily desaturated).
2. Place it in this folder.
3. It loads automatically via `style.css` (`.hero-atmosphere`).

Do **not** use busy scrolling character art; it hurts readability and looks dated.
